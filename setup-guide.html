✨ AI採点 セットアップガイド

ComPass Pro の AI自動採点を使えるようにする手順です

schedule 所要時間：約5分

1 Gemini APIキーを取得する

まず、GoogleのAI（Gemini）を使うための「鍵」を手に入れます。
完全無料で取得できます。

① リンクを開く

下のボタンをクリックして、Google AI Studio を開きます。

open_in_new Google AI Studio を開く
info
Googleアカウントでログインが必要です。普段使っているGoogleアカウントでOKです。
② APIキーを作成する

開いたページで「APIキーを作成」（Create API key）ボタンをクリックします。

touch_app 「Create API key」→ 「Create API key in new project」をクリック
③ APIキーをコピーする

表示されたAPIキー（AIza... で始まる長い文字列）をコピーします。

content_copy コピーアイコンをクリック → メモ帳に貼り付けて保存しておく
warning
このAPIキーは他の人に見せないでください。あなた専用の鍵です。

2 Google Apps Scriptにコードを貼る

次に、AIへの「橋渡し役」となるプログラムを設置します。
コードをコピペするだけです。プログラミングの知識は不要です。

① Apps Script を開く

下のボタンから Google Apps Script を開きます。

open_in_new Apps Script を開く（新規プロジェクト）
② コードを貼り付ける

エディタ画面が開くので：

select_all 最初から書いてあるコード（function myFunction(){}）を全部消す
content_paste gas-grader.js の中身を全部コピーして貼り付ける

gas-grader.js は ComPass Pro フォルダにあります。
メモ帳やVS Codeで開いて、全選択(Ctrl+A) → コピー(Ctrl+C) → 貼り付け(Ctrl+V)

③ プロジェクト名を変更する（任意）

左上の「無題のプロジェクト」をクリックして「ComPass Pro Grader」に変更。

④ 保存する
save Ctrl + S を押して保存（またはフロッピーアイコンをクリック）

3 APIキーを安全に登録する

Step 1で取得したAPIキーを、プログラムに安全に渡します。

① 設定画面を開く

Apps Script の左サイドバーにある⚙️歯車アイコン（プロジェクトの設定）をクリック。

settings 左メニューの一番下にある「プロジェクトの設定」をクリック
② スクリプトプロパティを追加

画面を一番下までスクロールして「スクリプト プロパティ」セクションを見つけます。

add_circle 「スクリプト プロパティを追加」をクリック

以下のように入力します：

プロパティ名 値
GEMINI_API_KEY Step 1でコピーしたAPIキーを貼り付け
save 「スクリプト プロパティを保存」をクリック

4 ウェブアプリとして公開する

作ったプログラムを、ComPass Proからアクセスできるように公開します。

① エディタに戻る

左サイドバーの< >（エディタ）をクリックしてコード画面に戻ります。

② デプロイをクリック

画面右上の青い「デプロイ」ボタンをクリック → 「新しいデプロイ」を選択。

rocket_launch 右上「デプロイ」→「新しいデプロイ」
③ 種類を選ぶ

「種類の選択」横の⚙️歯車アイコンをクリック → 「ウェブアプリ」を選択。

④ アクセス設定を変更する

「アクセスできるユーザー」を「全員」に変更します。

warning
「全員」に変更しないと、ComPass Proからアクセスできません。
必ず変更してください！
⑤ デプロイする
rocket_launch 「デプロイ」ボタンをクリック

初回は「アクセスを承認」を求められます：

person 自分のGoogleアカウントを選択 → 「詳細」→「～（安全でないページ）に移動」→「許可」
⑥ URLをコピーする

「ウェブアプリ URL」が表示されるので、コピーします。

https://script.google.com/macros/s/AKfycb.../exec
content_copy このURLをコピー！次のステップで使います

5 ComPass Proに接続する

最後のステップ！ComPass ProにURLを貼り付けるだけです。

① ComPass Proを開く

ComPass Proアプリを開いて、Step 2 または Step 3 に進みます。

② AI採点ボタンをクリック

緑色の「✨ AI採点」ボタンをクリックします。

初回はセットアップ画面が表示されます。

③ URLを貼り付けて保存

入力欄に、Step 4でコピーしたウェブアプリURLを貼り付けて「保存」をクリック。

check_circle
設定完了！🎉
もう一度「AI採点」を押すと、AIが要約を採点してくれます！

help よくある質問

💰 本当に無料ですか？

はい！Gemini APIの無料枠（1分あたり15回のリクエスト）で十分使えます。 高校のクラス全員が同時に使っても問題ありません。

🔒 APIキーは安全ですか？

はい。APIキーはGoogle Apps Script のサーバー側に保存されるため、 生徒や第三者からは見えません。ComPass Pro のコードにはAPIキーは含まれません。

❌ 「アクセスを承認」でエラーが出ます

「このアプリは Google で確認されていません」と表示されたら：
「詳細」をクリック → 「〜（安全ではないページ）に移動」をクリックしてください。 これは自分で作ったスクリプトなので安全です。

🔄 URLを変更したいときは？

ComPass Proでブラウザの開発者ツール（F12）→ Console に以下を入力：
localStorage.removeItem('writepass-gas-url')
その後「AI採点」を押すと再設定できます。

📊 採点基準は何ですか？

英語資格検定公式の4つの観点で採点します（各4点、合計16点満点）：
①内容 ②構成 ③語彙・パラフレーズ ④文法
